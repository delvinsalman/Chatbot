# AI Chatbot

A full-stack developed chatbot that integrates Gemini API models (2.0 Flash), providing users with a sophisticated platform for asking complex questions, image analysis, and customizable creativity levels. This modern chatbot system leverages heavy JavaScript programming alongside Flask frameworks, operating as a ChatGPT-like interface at a smaller, more developer-focused scale.

## 🚀 Features

### 🤖 AI-Powered Conversations
- **Gemini 2.0 Flash Integration**: Advanced AI model for intelligent responses
- **Real-time Messaging**: Instant AI responses with typing indicators
- **Contextual Understanding**: Maintains conversation context and history

### 🎨 Modern User Interface
- **Dark/Light Theme**: Toggle between themes with persistent settings
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Sidebar Navigation**: Easy access to conversation history and settings

### 📁 File Support & Analysis
- **Image Analysis**: Upload and analyze images using Gemini's vision capabilities
- **Multiple File Types**: Support for PDF, TXT, DOCX, and other document formats
- **File Previews**: Visual file attachments with preview capabilities

### ⚙️ Customization & Settings
- **Creativity Control**: Adjustable temperature slider (0.0-1.0) for response creativity
- **System Prompts**: Customizable AI behavior through system instructions
- **Conversation Management**: Rename, save, and organize chat sessions

### 💾 Advanced Features
- **Local Storage**: Conversations saved locally in browser storage
- **Search Functionality**: Find specific conversations quickly
- **Markdown Support**: Rich text formatting in AI responses with syntax highlighting
- **Fullscreen Mode**: Focused chatting experience

## 🛠️ Technology Stack

### Frontend
- **HTML5**: Semantic structure and accessibility
- **CSS3**: Modern styling with CSS variables and Flexbox/Grid
- **JavaScript ES6+**: Dynamic functionality and API interactions
- **Marked.js**: Markdown parsing for rich text responses
- **Highlight.js**: Code syntax highlighting

### Backend
- **Python Flask**: Lightweight web framework
- **Google Gemini API**: AI model integration
- **RESTful API**: Clean API design for frontend-backend communication

### Development Tools
- **Font Awesome**: Icon library
- **Google Fonts**: Inter font family
- **Local Storage**: Client-side data persistence
